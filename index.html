LibreHealth EHR

LibreHealth EHR is a project designed to be "easy to use" and "out of the box," so those with creative minds, capability in design, and skillsets in implementing components would be most efficient. For more information refer to "Read More"

Languages: php javascript mysql

LibreHealth Radiology

LibreHealth Radiology is a modified version of Toolkit with additional components for use alongside imaging proffesionals. This project requires creativity to design and implement new components. For more information refer to "Read More."

Languages: php javascript mysql CSS

LibreHealth Toolkit

LibreHealth Toolkit is the functional and foundational API for LibreHealth Radiology. It is suited for those who have had experience with code. This project requires the ability to design and implement new components. For more information refer to "Read More."

Languages: php javascript mysql CSS
LibreHealth EHR

LibreHealth EHR focuses on the patient experience, so user input from beginner's is key to the success of an electronic health record. LibreHealth EHR is run by a mysql server alongside an apache server, so you will be using the project on your own personal server. If you are interested in this branch first you should complete the task called "Become a part of the community" then move onto setup for Windows or your preferred system.

LibreHealth Radiology

LibreHealth Radiology requires a somewhat good understanding of programming in order to create new components to better the existing codebase, or to enhance the user interface from the patient dashboard to the design of the website. LibreHealth Radiology is a project that uses an enhanced version of LibreHealth Toolkit to eventually design a foundation for Radiologists and other imaging specialists to use electronically. If you are interested in this branch you should first do the task called "Become a part of the community" then follow up with the setup of Radiology for Windows or your preffered system.

LibreHealth Toolkit

LibreHealth Toolkit requires extensive coding to create new components from remodeling the patient dashboard to enhance user experience to changing the layout or themes of a tab such as fixing a container to fit when a screen is shrinked. LibreHealth Toolkit can also be used to design new wireframes for the website and patient experience. If you are interested in this branch you should first do the task called "Become a part of the community" then follow up with creating a video for the setup of LibreHealth Toolkit on Windows or your preferred system.
